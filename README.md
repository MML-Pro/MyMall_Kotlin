# My Mall

This is an open-sourced & full-stack E-Commerce App. I started this Project out of curiosity and absolutely no vision but now it has evolved into a full-size Shopping App. This repository holds the basic customer version of the app. You can download the demo app to see what it looks like now.
I am looking forward to making an Admin version of My Mall app,  If you are interested feel free to Ping me.

## Features:
  - Get All products and display them with three different types (View Pagger, Grid, Horizontal).
  - Get all details about the product, Product name, price, discounted Price, description, coupons, offers, and Rating.
  - Search for products.
  - Add, and delete products in the wish list.
  - Add, and delete products in the cart list.
  - Create a regular user (email & password) or sign in with one touch via Google, and save the user info.
  - Add new addresses, update and delete options.
  - Add and get product ratings
  - Making orders and making Secure payment via RazorPay payment Gate.
  - Show Order Status like Ordered, Packed, Shipped, Delivered in Orderes Section.
  - Show Order details information, the user order rate, ordered date, packed date, shipped date and delivered date, shipping details, username, and addresses, and finally the order amount summary.
  - Edit/Update the user info, Username, email, password
  - Support English and Arabic languages.
  - Support Dark and Light modes.
  - and much more
  
## Tech stack & Open-source libraries
- Minimum SDK level 21
- [Kotlin](https://kotlinlang.org/) based, [Coroutines](https://github.com/Kotlin/kotlinx.coroutines) together with [Flow](https://developer.android.com/kotlin/flow) for asynchronous streams and one side viewModel to fragment communication.
- Dagger Hilt for dependency injection.
- [Firebase Firestore](https://firebase.google.com/docs/firestore) for the backend database.
- [Firebase Authentication](https://firebase.google.com/docs/auth) for handling user auth creates/signing up and signing users.
- Datastore: to store key-value pairs or typed objects with protocol buffers.
- [Material 3 with XML](https://m3.material.io/) for the UI.
- [Glide](https://github.com/bumptech/glide): Glide is a fast and efficient open-source media management and image loading framework for Android that wraps media decoding, memory and disk caching, and resource pooling into a simple and easy-to-use interface.
  
  
- JetPack:
  - Fragments - Present a reusable portion of the app's UI.
  - ViewModel - UI-related data holder, lifecycle aware.
  - Navigation Component - Makes it easier to navigate between different screens and pass data in a type-safe way.

- Architecture:
  - [Android Clean Architecture (Onion-Arch)](https://www.raywenderlich.com/817602-mvi-architecture-for-android-tutorial-getting-started).
  - Repository pattern.
